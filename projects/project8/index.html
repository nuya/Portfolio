Took ownership of UX Design, User Research, UI Design, Frontend Engineering, Product Management, and implementing an Agile environment to increase sales by directing the User to a one page checkout in three clicks or less. While also adding animations and visuals to further increase a quality product.

Problem

The original website was information based and the company wanted to move towards a subscription based ecommerce platform and design.


Process

After studing analytics, recordings, and heatmaps I found that Users wanted to see the full product line and ingredients in the medicines the most.


Solution

First I stratigized and implemented a one page checkout because the checkout progress was to long as well as set up a new account management system so Users can manage their subscription. This allowed them to edit, swap, or cancel their subscription. If they wanted to cancel I set up a rebutal system to offer them incentives if they stay. Also, before the hero image was just an image with photoshoped buttons and text. I redeveloped it in HTML, CSS, and JavaScript to easily swap out the image and added animations. I set up components highlighting the subscription option, featured products, and a modal that when a User adds to cart it shows with the option to direct them to the one page checkout. A product grid component that features some of the ingredients was set up that navigates popular products, product categories, and new products so the User can easily navigate the entire product line. These changes lead to around an 80% increase in sales for each month.
