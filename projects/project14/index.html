  • Travel App Challenge


    UX/UI Design

  • Role
    UX/UI Designer

Problem

There is a lot of infomation and elements crammed into a small space that can be intimidating for a User to navigate and in not very intuative. The User may only be interested in a certain section or feature.


Process

Arrange features, elements, and articles in a minimalist UI focusing on each as a separate entity so Users can easily focus on a specific target while at the same time highlighting more features of the app.


Solution

With a more intuative UI Users can find find what they are looking for faster and easier. This will lead to lesser dropoffs and and an increase in User interaction.
